# Changelog

## Unreleased — Instituto Travessia headquarters

### Added

- Primary headquarters startup and return loop.
- Six route authorizations with deterministic parameter profiles.
- Persistent profile migration, field rank, scrip, run statistics, and remote-object totals.
- Persistent object archive with appraisal, provenance, condition, route identity, local/remote seeds, storage, sale, retention, and buyback.
- Seven permanent institutional permissions.
- Headquarters route board, archive, profile, and permissions interface.
- Temporary original Instituto Travessia and A Chave Geral vector marks.
- Asset manifest separating temporary systems assets from future final art.
- Headquarters economy simulation check in CI.
- Headquarters implementation and decision documentation.

### Changed

- The game now begins at Instituto Travessia rather than immediately opening a field briefing.
- Route selection generates the mission and then opens its briefing.
- Debrief returns to headquarters instead of directly generating another run.
- Profile persistence moved to version 2 with legacy migration.
- Institute retention is now a deterministic per-object roll capped at fifteen percent of each filing.
- Route payout, remote-object value, healing, cooldown, safe-window, seizure, remote-vault, and Auditor behavior can be changed by permanent permissions.

## 0.2.0 — Independent interlace

### Added

- Independently seeded remote room graph.
- Cross-state overlap detection and temporary bridges.
- Traversable remote rooms and corridors.
- Remote combat, shrines, vault, loot, provenance, and contract objectives.
- Dual-state minimap.
- Threshold seizure pressure and remote-object consequences.
- Multi-seed deterministic generation checks.

## 0.1.0 — Playable field loop

### Added

- Four operative kits: Sócrates, Zélia Amato, Lia, and Kindred.
- Combat, abilities, dodges, hazards, loot, contracts, interlace, extraction, payout, and local progression.
- Pursuer, gunner, elite, and Auditor enemy processes.
- Tactical HUD, briefing, minimap, and debrief.
